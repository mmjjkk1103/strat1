
# Project Blueprint

## Overview

This project is a simple, visually appealing lottery number generator. It provides a clean interface for users to generate a set of unique random numbers for their lottery tickets.

## Current State & Features

*   **Initial Project:** The project started as a basic "Hello, World!" HTML page.
*   **Lottery Generator:**
    *   **UI:** A central card displays the title, the generated numbers, and a "Generate Numbers" button.
    *   **Functionality:** Clicking the button generates 6 unique random numbers between 1 and 45.
    *   **Styling:** Modern CSS with a clean layout, custom fonts, drop shadows for a "lifted" effect, and a subtle background texture. The color palette is vibrant and engaging.
*   **Localization:** The user interface has been translated into Korean.
